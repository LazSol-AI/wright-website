Frequently Asked Questions

How do I set up my API keys?

Visit our API Key Setup Guide for detailed instructions on how to set up and secure your API keys.

Which LLM services are supported?

Wright currently supports:

  • OpenAI (ChatGPT)
  • Groq
  • Grok (xAI)
  • DeepSeek
  • Gemini (Google)

Is my data secure?

Yes! Wright does not collect or store any of your data. Your API keys are securely stored in your macOS Keychain. Read our Privacy Policy for more details.
